# GitHub-Explorer
A web app that allows users to search GitHub repositories, filter them by language, and sort them based on stars or forks. Users can also save favorite repositories for later.

##  Project Description

The GitHub Repository Explorer is a web application that allows users to search for GitHub repositories using keywords. It provides features such as filtering by programming language, sorting repositories based on stars or forks, and saving favorite repositories.

##  Features

* Search repositories using GitHub API
* Filter repositories by language
* Sort repositories by stars and forks
* Save favorite repositories using local storage
* Responsive user interface

##  API Used

GitHub REST API
Endpoint: https://api.github.com/search/repositories

##  Technologies Used

* HTML
* CSS
* JavaScript (Fetch API, Array Higher Order Functions)
